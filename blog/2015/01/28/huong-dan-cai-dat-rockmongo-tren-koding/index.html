Hướng dẫn cài đặt RockMongo trên Koding

Sau khi cài đặt MongoDB, chắc hẳn bạn sẽ khó thao tác với nó trên màn hình Terminal. Để giải quyết vấn đề này, ta cần sử dụng giao diện để dễ dàng thao tác với nó. Hôm nay, mình sẽ hướng dẫn các bạn cách sử dụng RockMongo để thao tác với MongoDB.

Thật ra Koding cũng có một tutorial được viết bằng tiếng Anh. Nhưng để tiện cho những ai vốn tiếng Anh không được khá nên mình viết lại bằng tiếng Việt.

RockMongo

RockMongo là một công cụ quản trị dành cho MongoDB, được viết bằng PHP 5.
Do trang thủ của RockMongo là http://rockmongo.com không vô được, nên các bạn vui lòng coi chi tiết trên Github.

Cài đặt

Đầu tiên, bạn sử dụng quyền Root và vào thư mục /home/USER_NAME>/Web.

Do trên Koding đã cài sẵn Git nên bạn cứ việc clone phiên bản mới nhất về từ Github.
git clone --depth=1 https://github.com/iwind/rockmongo.git

Sau đó bạn mở file /home/USER_NAME>/Web/config.php để cấu hình RockMongo. Theo Koding đề nghị thì bạn nên cấu hình như sau.

  • $MONGO["servers"][$i]["mongo_host"] - nên đặt là 0.0.0.0, để 127.0.0.1 cũng được.
  • $MONGO["servers"][$i]["mongo_port"] - Tùy bạn, mình vẫn để mặc định là 27017.
  • Username và Password - Mặc định là "admin" - "admin".
    $MONGO["servers"][$i]["control_users"]["admin"] = "admin";

Sau đó bạn cài PHP Driver bằng lệnh pecl install mongo. Nếu có hiển thị như hình bên dưới cũng không sao vì nó đã có sẵn trên máy.

Nếu nó chưa có và cài đặt thành công thì bạn vào file /etc/php5/apache2/php.ini để thêm driver vào.
Bạn gõ nano /etc/php5/apache2/php.ini để mở file. Sau đó bấm Ctrl + W để tìm kiếm từ khóa "Dynamic Extension" di chuyển con trỏ xuống phía dưới và thêm dòng extension=mongo.so.

Sau đó bấm Ctrl + X -> Y -> Enter để lưu lại. Rồi bạn restart lại Apache server bằng lệnh service apache2 restart.

Bây giờ, bạn có thể vào RockMongo với đường dẫn trên URL là http://YOUR_KODING_USERNAME.koding.io/rockmongo với tài khoản đã cấu hình ở bước trên.
