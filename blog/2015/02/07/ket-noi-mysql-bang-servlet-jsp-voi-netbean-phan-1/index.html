Kết nối MySQL bằng Servlet - JSP với NetBean (phần 1)

Cấu hình môi trường để chạy được web với ngôn ngữ Java có phần phức tạp so với những ngôn ngữ mình đã làm qua. Bài viết này hơi nhiều, mình xin chia ra làm 2 phần. Ở phần 1, mình chủ yếu trình bày cách cấu hình MySQL trong NetBean để tiện thao tác, đỡ phải mở một chương trình khác để thao tác. Nếu bạn có thể chuyển sang phần 2 để coi cách kết nối bằng ngôn ngữ Java.

MySQL

MySQL là hệ quản trị cơ sở dữ liệu mã nguồn mở phổ biến nhất thế giới, thuộc loại cơ sở dữ liệu quan hệ (RDBMS) và được các nhà phát triển rất ưa chuộng trong quá trình phát triển ứng dụng. Vì MySQL là cơ sở dữ liệu tốc độ cao, ổn định và dễ sử dụng, có tính khả chuyển, hoạt động trên nhiều hệ điều hành cung cấp một hệ thống lớn các hàm tiện ích rất mạnh. Với tốc độ và tính bảo mật cao, MySQL rất thích hợp cho các ứng dụng có truy cập CSDL trên internet.

MySQL được sử dụng cho việc bổ trợ PHP, Perl, và nhiều ngôn ngữ khác, nó làm nơi lưu trữ những thông tin trên các trang web viết bằng PHP hay Perl,...

Cấu hình

Bạn khởi động NetBean, ở góc trên trái, chọn thẻ Services -> Database, chuột phải chọn New Connection.

Ở mục Driver chọn MySQL (Connector/J driver). Ở khung Driver file(s) sẽ hiển thị driver tương ứng. Lúc mình cài NetBean 8.0, nó đã có sẵn nên không cần phải download. Nhấn Next

Mục host thì mình không thích chữ localhost nên đổi thành 127.0.0.1 cho đẹp, chứ thật ra tụi nó là một. Chọn port 3306 vì đây là port mặc định của MySQL. Mục Database thì nhập tên của database mà bạn muốn sử dụng. User Name, Password là của tài khoản để vào phần quản trị MySQL (tài khoản vào PHPMyAdmin cho dễ hiểu). Xong rồi nhấn Test Connection để kiểm tra có kết nối thành công hay không.

Chỗ JDBC URL là url dùng để kết nối tới CSDL. Cấu trúc của nó gồm jdbc:mysql:// + <host> + : + <port> + ⁄ + <database>.

Sau khi nhấn Next ta sẽ tới bước Choose Database Schema. Bước này cứ mạnh dạn Next để tới Choose name for connection. Ở khung Input connection name, nếu không thích tên xấu xí thì đổi.

Cuối cùng Finish.

Bạn có thể thao tác thêm/xóa/sửa/truy vấn ngay tại NetBean. Và lúc tạo bảng, mặc định NetBean sẽ chọn InnoDB.

Đã hết phần 1, các bạn có thể coi tiếp phần 2 tại đây.
