Hướng dẫn thao tác với repository trên GitHub

Chức năng chính của GitHub dùng để lưu trữ source code. Bài viết này, mình sẽ giúp các bạn thao tác với một repository trên GitHub. Có một số vấn đề trong bài viết này, mình sẽ sử dụng TortoiseGit để thao tác với GitHub và sử dụng giao thức SSH để truyền dữ liệu.

Tạo SSH Key

Bạn mở PuTTYgen để bắt đầu tạo SSH Key. Lưu ý là mã được tạo ra bằng thuật toán RSA. Do đó, bạn sẽ cần lưu 2 khóa public và private.

Tiếp theo, bạn nhấn nút Generate và di chuyển con chuột trong vùng được khoanh màu đỏ để tạo SSH Key.

Sau khi hoàn tất, bạn nhấn Save private key để lưu khóa private. Tiếp theo, bạn copy đoạn trong khung được khoanh tròn để lưu lại khóa public.

TortoiseGit

Đầu tiên, bạn phải vào trang http://git-scm.com/download/win để tải Git về và cài đặt.

Tiếp theo, vào trang https://code.google.com/p/tortoisegit/wiki/Download để tải TortoiseGit rồi cài. Sau khi cài xong, bạn nhấn chuột phải sẽ ra như hình bên dưới.

Thêm SSH Key vào GitHub

Bạn vào phần Settings trên GitHub. Nhìn sang cột bên trái, chọn SSH Keys.

Nhấn nút Add Key, điền Title và copy Public Key vào khung Key.

Nhấn nút Add key để nó điều hướng sang trang khác. Xác nhận password là xong.

Tạo repository

Sau khi tạo xong SSH Key, cài đặt Git và TortoiseGit xong, chúng ta sẽ tạo một repository trên GitHub.

Ta vào trang chủ trên GitHub, chọn tab Repositories, nhấn nút New.

Bạn điền đẩy đủ thông tin bao gồm tên của repo, mô tả (description), public hay private, có cần file README hay không, chịu ảnh hưởng của bản quyền mã nguồn mở nào,...

Và đây là kết quả.
