Hướng dẫn thay đổi Docroot trên GlassFish 4

Ngày xưa, lúc mới học PHP, mình rất ghét khi nhìn URL khi đang ở root mà phải ở dạng 127.0.0.1:8080/xxx thay vì 127.0.0.1:8080/ khi sử dụng WAMP Server, vì thế mình sử dụng VertrigoServ. Bữa nay làm quen với JavaEE, lại gặp vấn đề tương tự khi sử dụng GlassFish 4.

GlashFish

GlassFish là một Application Server mã nguồn mở. Ban đầu, dự án này được phát triển bởi Sun Microsystems cho nền tảng JavaEE và bây giờ là Oracle Corporation. GlashFish là một phần mềm miễn phí, chịu ảnh hưởng bởi 2 licence là Common Development and Distribution License (CDDL) và GNU General Public License (GPL).

Phiên bản mới nhất hiện tại là 4.1.

Giới thiệu

Bạn chuẩn bị một source code để thử. Nếu chưa có thì làm theo ở đây.

GlassFish chạy mặc định ở port 8080. Và mặc định, khi bạn gõ 127.0.0.1:8080 nó sẽ ra trang mặc định của GlashFish.

Và bạn phải gõ 127.0.0.1:8080/First để tới project của mình.

Cấu hình

Đầu tiên, bạn phải vào GlassFish Console thông qua đường dẫn 127.0.0.1:4848.

Bạn nhìn vào cột Tree -> Configurations -> server-config -> Virtual Servers -> server.

Ta chỉnh lại Default Web Module để chọn docroot là project của mình, ở đây là project First.

Nhấn nút Save ở góc phải phía trên. Kéo lên trên, ta thấy thông báo "New values successfully saved" coi như là thành công.

Bây giờ, ta mạnh dạn vào đường dẫn 127.0.0.1:8080 để coi kết quả.

Lưu ý

Trong quá trình sử dụng, nếu GlassFish có vấn đề khi chạy với docroot thì bạn chịu khó chỉnh nó về default và restart lại nhé.
